# AI Customer Support Agent Platform

A full-stack AI-powered customer support platform built using React, TypeScript, FastAPI, and SQLite.

## Features

- AI-style customer support workflow
- Ticket creation and management
- Ticket priority classification
- Chat history management
- Knowledge base management
- Ticket filtering and search
- Admin dashboard
- REST API backend
- React + TypeScript frontend

## Tech Stack

### Frontend
- React
- TypeScript
- Axios
- CSS

### Backend
- FastAPI
- Python
- SQLAlchemy
- SQLite

## Project Architecture

Frontend (React)
↓
FastAPI Backend
↓
AI Workflow Logic
↓
SQLite Database

## How to Run

### Backend

cd backend
uvicorn main:app --reload

### Frontend

cd frontend
npm install
npm run dev
